# Chuyên đề tốt nghiệp 2 - Pharmacy Online

Đây là kho code của gồm các thành viên:
- Lê Văn Tùng
- Lý Ngọc Quân

## Giới thiệu
Dự án quản lý nhà thuốc (Pharmacy Admin) sử dụng Flutter và Firebase.

## Tính năng
- Quản lý kho hàng (Inventory)
- Bán hàng tại quầy (POS)
- Quản lý đơn hàng online
- Thống kê doanh thu
